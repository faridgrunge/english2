Contoh Kalimat 16 Tenses

 

Seperti yang sudah dijelaskan sebelumnya bahwa, secara garis besar, Tenses dikelompokkan menjadi empat bagian, yakni:

  • Past tense (masa lampau)
  • Present tense (masa kini)
  • Present Future tense (masa akan datang)
  • Past Future tense (masa lampau yang akan datang)

Dalam satu tense masing-masing terbagi menjadi empat yakni:

  • Simple
  • Continuous/Progressive
  • Perfect
  • Perfect Continuous/Progressive

Berikut ini contoh penggunaan ke 16 tenses dalam kalimat dengan menggunakan kata kerja yang sama.

Kali ini kita coba menggunakan kata "menulis" = write (V1) - wrote (V2) - written (V3)

  • A1 = Simple Past (V2)

Contoh dan Rumus :

  • (+) I wrote a book (S + V2 + O)
  • (-) I did not write a book (S+did not + V1 + O)
  • (?) Did I write a book? (Did +S +V1 + O ?)
  • A2 = Simple Present (V1)

Contoh dan Rumus :

  • (+) I write a book (S + V1(s/es) + O)
  • (-) I do not write a book (S+do/does not + V1 + O)
  • (?) Do I write a book (Do/does +S +V1 + O ?)
Penggunaan "does" jika subjeknya orang ke-3 tunggal: he/she/it
  • (+) He writes a book (S + V1(s/es) + O)
  • (-) He does not write a book (S+do/does not + V1 + O)
  • (?) Does He write a book (Do/does +S +V1 + O ?)
  • A3 =Simple Present Future (will+V1)

Contoh dan Rumus :

  • (+) I will write a book (S + will + V1 + O)
  • (-) I will not write a book (S+will not + V1 + O)
  • (?) Will I write a book ?(Will +S +V1 + O ?)
  • A4 =Simple Past Future (would+V1)

    Contoh dan Rumus :

  • (+) I would write a book (S + would + V1+ O )
  • (-) I would not write a book (S + woulld not + V1 + O)
  • (?) Would I write a book (Would +S +V1 + O ?)

 

  • B1 = Past Continuous (was/were+V1 ing)

    Contoh dan Rumus :

  • (+) I was writing a book (S + was/were+V1 ing+O)
  • (-) I was not writing a book (S + was/were+not+V1 ing+O)
  • (?) Was I writing a book (Was/were + S +V1 ing+O ?)
  • (+) You were writing a book (S + was/were+V1 ing+O)
  • (-) You were not writing a book (S + was/were+not+V1 ing+O)
  • (?) Were You writing a book (Was/were + S +V1 ing+O ?)
  • B2 = Present Continuous (am/is/are+V1 ing)

    Contoh dan Rumus :

  • (+) I am writing a book (S + am/is/are +V1 ing+O)
  • (-) I am not writing a book (S + am/is/are +not+V1 ing+O)
  • (?) Am I writing a book (Am/is/are + S +V1 ing+O ?)
  • B3 = Present Future Continuous (will be + V1 ing)

    Contoh dan Rumus :

  • (+) I will be writing a book (S + will + be+ V1ing + O)
  • (-) I will not be writing a book (S+will not + be +V1ing + O)
  • (?) Will I be writing a book (Will +S +be + V1ing+ O ?)
  • B4 = Past Future Continuous (would be + V1 ing)

    Contoh dan Rumus :

  • (+) I would be writing a book (S + would + be + V1ing + O)
  • (-) I would not be writing a book (S + woulld not + be + V1ing + O)
  • (?) Would I be writing a book (Would +S + be + V1ing + O ?)
  • C1 = Past Perfect (had+ V3)

    Contoh dan Rumus :

    • (+) I had written a book (S + had+ V3+ O)
    • (-) I had not written a book (S + had not + V3 + O)

    • (?) Had I written abook (Had +S +V3 + O ?)

  • C2 = Present Perfect (has/have+V3)

    Contoh dan Rumus:

    • (+) I have written a book (S + has/have+ V3+ O)
    • (-) I have not written a book (S + has/have not + V3 + O)
    • (?) Have I written a book (Has/Have+S +V3 + O ?)
    • (+) She has written a book (S + has/have+ V3+ O)
    • (-) She has not written a book (S + has/have not + V3 + O)
    • (?) Has she written a book (Has/Have+S +V3 + O ?)
  • C3 = Present Future Perfect (will + have+ V3)

    Contoh dan Rumus :

    • (+) I will have written a book (S + will + have+ V3+ O)
    • (-) I will not have written a book (S + will + have not + V3 + O)
    • (?) Will I have written a book ? (Will+S + have + V3 + O ?)
  • C4 = Past Future Perfect (would + have+ been+V1 ing)

    Contoh dan Rumus :

      • (+) I would have been writing a book (S + would+ have+ been + V1ing+ O)
      • (-) I would not have been writing a book (S +would+ have+ been + V1ing+ O)
      • (?) Would I have been writing a book ? (Would +S + have + been + V1ing + O ?)
  • D1 = Past Perfect Continuous (had+been+V1 ing)

    Contoh dan Rumus :

      • (+) I had been writing a book ( S + had+ been + V1ing+ O)
      • (-) I had been writing a book (S + had not + been + V1ing+ O)
      • (?) Had I been writing a book ? (Had +S + been + V1ing+ O?)
  • D2 = Present Perfect Continuous (has/have+been+V1 ing)

    Contoh dan Rumus :

      • (+) I have been writing a book (S + has/have+ been + V1ing+ O)
      • (-) I have been writing a book (S + has/have + not + been + V1ing+ O)
      • (?) Have I been writing a book ? (Has/Have+S + been + V1ing+ O?)
      • (+) He has been writing a book (S + has/have+ been + V1ing+ O)
      • (-) He has been writing a book (S + has/have + not + been + V1ing+ O)
      • (?) Has he been writing a book ? (Has/Have+S + been + V1ing+ O?)

       

  • D3 = Present Future Perfect Continuous (will+have+been+V1 ing)

    Contoh dan Rumus :

    • (+) I will have been writing a book (S + will + have+ been + V1ing + O)
    • (-) I will not have been writing a book (S + will + have not + V1ing + O)
    • (?) Will I have been writing a book ? (Will+S + have + been + V1ing+ O?)
  • D4 =Past Future Perfect Continuous (would+have+been+V1 ing)

    Contoh dan Rumus :

    • (+) I would have been writing a book (S + would + have+ been + V1ing + O)
    • (-) I would not have been writing a book (S + would+ have not + V1ing + O)
    • (?) Would I have been writing a book? (Would+S + have + been + V1ing+ O?)
