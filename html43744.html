Lost

(Tersesat)

 

English

Linda : Are you hungry?
Robert : No, why?

Linda : We've been driving for a long time. I think we'd better find some place to eat soon.
Robert : No, I'm OK. We're late so we'd better not stop.

Linda : We've passed that same store three times already. I think we're lost. We should ask someone for directions.
Robert : We're not lost.

Linda : I've never gone this way before. Do you know where you're going?
Robert : Yes, this road goes through town. It's faster than the highway.

Linda : OK. Would you mind driving a little slower please?
Robert : Sure, no problem.

Linda : Did you see that sign back there? I think this is a one way street.
Robert : No, I didn't see it. I'm going to have to turn around.

Linda : You can't turn around here. I think you should let me drive.
Robert : I think you're right. I'm getting tired.

Indonesian

Linda: Apakah Anda lapar?
Robert: Tidak, kenapa?

Linda: Kita sudah mengemudi lama sekali. Sebaiknya kita cari tempat untuk makan sebentar lagi.
Robert: Tidak, saya tidak apa-apa. Kita terlambat jadi sebaiknya tidak berhenti.

Linda: Kita sudah melewati toko yang sama sebanyak tiga kali. Saya kira kita tersesat. Sebaiknya kita tanya arah kepada seseorang.
Robert: Kita tidak tersesat.

Linda: Saya tidak pernah lewat jalan ini sebelumnya. Anda tahu jalan?
Robert: Ya, jalan ini melewati kota. Lebih cepat dibandingkan jalan tol.

Linda: Oke. Bisa mengemudi lebih lambat?
Robert: Tentu saja, tidak masalah.

Linda: Apakah Anda lihat tanda di belakang tadi? Saya pikir ini jalan satu arah.
Robert: Tidak, saya tidak lihat. Saya harus putar.

Linda: Anda tidak bisa putar di sini. Saya pikir Anda harus biarkan saya saja yang menyetir.
Robert: Saya kira Anda betul. Saya mulai capek.
