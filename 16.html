Percakapan Bahasa Inggris Singkat

 

Contoh Percakapan dengan Turis di Tempat Wisata

Santi                 : Hello, good afternoon, Sir. (Halo, selamat siang Pak).
Mr. Robert     : Hello, good afternoon. (Halo, selamat siang).
Santi                 : May I talk with u for a while, Sir? (Bolehkah saya berbicara sebentar dengan  anda Pak?).
Mr. Robert     : Yes, of course. (Ya, tentu saja).
Santi                 : My name is Santi. What is your name, Sir? (Nama saya Santi. Siapa nama anda Pak?).
Mr. Robert     : Hi Santi, my name is Robert. (Hi Santi, nama saya Robert).

Santi                 : Where are you from Mr. Robert? (Darimana anda berasal Pak Robert?).

Mr. Robert     : I am from New York, America. (Saya berasal dari New York, Amerika).

Santi                 : Wow, so far away from here. How long have you been here Mr. Robert?  (Wow, sangat jauh dari sini. Sudah berapa lama anda berada disini Pak Robert?).

Mr. Robert     : I have been here for three days. This is my second time in Bali. (Saya sudah  berada disini selama tiga hari. Ini adalah kedua kalinya saya berada di Bali).

Santi                 : Is this your first time to visit Garuda Wisnu Kencana Mr. Robert? (Apakah  ini pertama kalinya anda mengunjungi Garuda Wisnu Kencana Pak Robert?).

Mr. Robert     : Yeah, this is my first time to visit this place. I really want to come here.  (Ya, ini adalah pertama kali saya mengunjungi tempat ini. Saya sangat ingin datang kemari).

Santi                 : So, what do you think about this place Mr. Robert? (Jadi, apa yang anda pikirkan tentang tempat ini Pak Robert?).

Mr. Robert     : I think this place is very interesting. I was amazed by the statue.  (Saya pikir  tempat ini sangat menarik. Saya kagum dengan patungnya).

Santi                 : Yes, It is so high. Most of the visitors are amazed by the statue. (Ya, patungnya begitu tinggi. Kebanyakan pengunjung kagum dengan patungnya).

Mr. Robert     : Do you know how high it is? (Apakah kamu tahu seberapa tingginya?).

Santi                 : Yes, it is about 23 meters. (Ya, tingginya sekitar 23 meter).

Mr. Robert     : Really? That’s awesome! What is the name of the statue? (Benarkah? Betapa mengagumkan! Apakah nama patung tersebut?).

Santi                 : We called it Lord Wisnu. (Kami menyebutnya Dewa Wisnu).

Mr. Robert     : Oh, I see. So it symbolizes the God. (Oh, begitu. Jadi patungnya menyimbolkan Tuhan)

Santi                 : Yes, Mr. Robert. By the way, thanks for your time. (Ya, Pak Robert. Ngomong –ngomong, terima kasih atas waktunya).

Mr. Robert     : Yes, thanks for your additional information about this place. (Ya, terima kasih atas informasi tambahanmu mengenai tempat ini).

Santi                 : Nice to meet you Mr. Robert. (Senang bertemu denganmu Pak Robert).

Mr. Robert     : Nice to meet you too Santi. (Senang bertemu denganmu juga Santi).

Percakapan bahasa inggris perkenalan dengan turis ( Perkenalan )

Alisa : Good morning, Sir. ( Selamat pagi pak )
Robert : Good morning.( Selamat pagi )
Alisa : May I have time to talk with you? ( Bolehkah saya bicara sebentar dengan mu )
Robert : Yes, sure. What can I do for you? ( Tentu, apa yang bisa saya perbuat untukmu )
Alisa : I am Alisa, I have a task to write a diary about meeting foreigners. ( Saya Alisa, saya punya tugas untuk menulis tentang pertemuan dengan para turis )
Robert : Mmm, I am Robert, I came from England. ( Mmm, saya robert, saya datang dari England )
Alisa : Nice to meet you, Sir. ( Senang bertemu denganmu pak )
Robert : Nice to meet you too. ( Senang juga bertemu denganmu )

Dialog di atas adalah contoh perkenalan dengan turis. Akan lebih baik lagi jika ditambahkan dengan beberapa percakapan lain agar si turis merasa senang bisa berbincang dengan anda. Misalnya, tambahkan:

Alisa : May I ask something, Sir? ( Bolehkah saya menanyakan sesuatu pak )
Robert : Yes, sure. Why not. ( Ya boleh. Kenapa tidak )
Alisa : How do you think about this temple? ( Bagaimana pendapatmu tentang candi ini )
Robert : This is the second temple I've ever visited. Although it is not so big as Borobudur, I think it is also incredible place I know. ( Ini adalah candi kedua yang pernah saya kunjungi. Meski tidak begitu besar seperti Borobudur, saya rasa ini juga tempat yang luar biasa yang saya tahu )
Alisa : How about Indonesian people, Sir? ( Bagaimana pendapatmu tentang Indonesia pak )
Robert : Indonesian people are very kind as like you. They welcome us very nicely. I am happy English is still learned by most people in this country. ( Orang Indonesia sangat baik seperti kamu. Mereka menyambut kami dengan sangat baik. Saya senang bahasa Inggris masih dipelajari oleh kebanyakan orang di negeri ini )
Alisa : Thank you, Sir. I am just practicing my English. How long have you been in Indonesia, Sir? ( Terima kasih Pak. Saya hanya berlatih bahasa Inggris saya. Sudah berapa lama kamu di Indonesia Pak? )
Robert : I have been here for 7 days. I have visited many places in this country. ( Saya telah berada di sini selama 7 hari. Saya telah mengunjungi banyak tempat di negara ini. )
Alisa : What places have you visited, Sir? ( Tempat apa yang pernah kamu kunjungi, Pak? )
Robert : There are a lot of places, such as Bali, Lombok, Mount Bromo, Jogja Palace, Borobudur, this temple, and so on. I enjoyed this vacation. ( Ada banyak tempat, seperti Bali, Lombok, Gunung Bromo, Jogja Palace, Borobudur, candi ini, dan sebagainya. Saya menikmati liburan ini. )
Alisa : I hope you will come to Indonesia again next time. ( Saya harap kamu akan datang ke indonesia lagi lain kali. )
Robert : Yes, I will. ( Ya saya mau )
Alisa : Thanks for your time talking with you, Sir. It is pleasure to meet you. ( Terima kasih sudah waktumu berbicara denganmu, Pak. Senang bertemu dengan Anda. )
Robert : You're welcome, I am glad to see you, Alisa. ( Selamat datang, saya senang bertemu denganmu, Alisa. )
