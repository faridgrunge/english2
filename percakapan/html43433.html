Meeting at restaurant

(Bertemu di restoran )

 

English

Charles : Jennifer, would you like to have dinner with me?
Jennifer : Yes. That would be nice. When do you want to go?

Charles : Is today OK?
Jennifer : Sorry, I can't go today.

Charles : How about tomorrow night?
Jennifer : Ok. What time?

Charles : Is 9:00PM all right?
Jennifer : I think that's too late.

Charles : Is 6:00PM OK?
Jennifer : Yes, that's good. Where would you like to go?

Charles : The Italian restaurant on 5th street.
Jennifer : Oh, I don't like that Restaurant. I don't want to go there.

Charles : How about the Korean restaurant next to it?
Jennifer : OK, I like that place.

 

Indonesian

Charles: Jennifer, apakah Anda mau makan malam dengan saya?
Jennifer: Ya. Tentu saja. Kapan Anda mau?

Charles: Apakah hari ini oke?
Jennifer: Maaf, saya tidak bisa pergi hari ini.

Charles: Bagaimana kalau besok malam?
Jennifer: Oke. Jam berapa?

Charles: Apakah jam sembilan malam baik untuk Anda?
Jennifer: Saya kira itu terlalu malam.

Charles: Apakah jam enam sore oke?
Jennifer: Ya, baik. Mau ke mana?

Charles: Restoran Italia di "5th Street"
Jennifer: Aduh, saya kurang suka restoran itu. Saya tidak mau pergi ke sana.

Charles: Bagaimana kalau restoran Korea di sebelahnya?
Jennifer: Oke, saya suka tempat itu.
